Initial concepts

  • Observable - Some future stream of values or events
  • Observer - A callback/callbacks that listens to events by obserable
  • Subscription - The execution itself (a value returned from subscribe action)
  • Operators - Pure functions that react to the stream of data and alter it
  • Subject - A special multicast observable
  • Schedulers - Sychronization mechanism for observables
